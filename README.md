# OCR Text Replacer (Windows Desktop App)

이 프로그램은 이미지 내의 텍스트를 감지하고, 사용자가 입력한 새로운 텍스트로 해당 위치를 채워주는(stretching) 윈도우 데스크탑 애플리케이션입니다.

## 기능
1. **이미지 열기**: JPG, PNG 등 이미지 파일을 불러옵니다.
2. **OCR 분석**: 이미지 내의 한글 및 영어 텍스트를 자동으로 감지합니다.
3. **텍스트 편집**: 감지된 텍스트 영역을 클릭하여 내용을 수정합니다.
4. **Stretch 적용**: 수정된 텍스트를 원래 영역의 크기에 딱 맞게 늘려서(Stretch) 덮어씁니다.
5. **이미지 저장**: 수정된 이미지를 저장합니다.

## 설치 방법

1. Python이 설치되어 있어야 합니다. (3.8 이상 권장)
2. 필요한 라이브러리를 설치합니다.
   ```bash
   pip install -r requirements.txt
   ```
   *참고: `easyocr` 설치 시 PyTorch가 함께 설치되므로 시간이 조금 걸릴 수 있습니다.*

## 실행 방법

```bash
python main.py
```

## 사용법

1. `이미지 열기` 버튼을 눌러 이미지를 불러옵니다.
2. `OCR 분석 실행` 버튼을 누릅니다. (처음 실행 시 모델 다운로드로 시간이 걸릴 수 있습니다.)
3. 이미지 위에 빨간색 박스가 표시되면, 수정하고 싶은 박스를 클릭합니다.
4. 우측 패널의 '변경할 텍스트' 입력란에 원하는 글자를 입력합니다.
5. `적용 (Stretch)` 버튼을 누르면 이미지에 바로 반영됩니다.
6. 작업이 끝나면 `이미지 저장` 버튼을 눌러 저장합니다.

## 주의사항
- 배경 지우기 기능은 현재 단순하게 구현되어 있어(좌상단 픽셀 색상으로 덮기), 복잡한 배경에서는 티가 날 수 있습니다.
- 폰트는 윈도우 기본 한글 폰트인 `malgun.ttf`를 사용합니다.
